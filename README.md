# @amlplugins/stability-three-d

Native `stability-ai` SDK re-exported under the `@amlplugins` namespace.

> Stability AI 3D — Stable Fast 3D image-to-3D mesh generation (.glb).

## Install

```bash
npm install @amlplugins/stability-three-d
```

## Use

```ts
import { /* named exports from stability-ai */ } from "@amlplugins/stability-three-d";
// or
import sdk from "@amlplugins/stability-three-d";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
